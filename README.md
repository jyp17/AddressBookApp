AddressBookApp is an Android application that lists contacts based on user input.

TO RUN THE APP (using an emulator in Android Studio):
1. Launch Android Studio.

2. Open an existing project and select the folder containing the AddressBook app.

3. If you have not already, create an Android Virtual Device (AVD) by going to Tools > AVD Manager and click "Create Virtual Device". Select the device and system image you want. For reference, this app was tested on Pixel XL and Pixel 3a, using API levels 30 and 23 for both devices. It is recommended to run an emulator using these settings for reliability.

4. Once you have an AVD set up, confirm that you your app selected from the configurations menu in the toolbar. Next, select your desired AVD from the device menu.

5. Click the Run button to launch the app.


TO USE THE APP:
1. Click on the text field to type the first name of the contact you wish to add.

2. Click on the "Add" button to add the contact to the address book.

3. A list of contact names that have been added will appear below the text field. Presently, only the first name of the contacts that you have typed and added will appear on the list.
